# DImodelsMulti
An R package which acts as an add-on to the DImodels package, enabling the fitting of multivariate repeated measures Diversity-Interactions models.
The function DImulti() is the main method of the package, working similarly to DImodels::DI(). It outputs an object of custom class DImulti.
The package contains five example datasets, two real-world (dataSWE, dataBEL) and three simulated (simMV, simRM, simMVRM).
